Application web

Pour entrer dans l'application vous devez vous identifier. Cette application permet de connaître l'opinion des membres de l'ACCQ. À l'intérieur vous verrez une liste de carte représentant les enjeux à évaluer. Les notes sont de 1 à 5, 1 voulant dire que vous être totalement contre, 5 que vous êtes totalement en accord, 3 que vous êtes ambivalent, etc.. Tout le monde peut voir la moyenne des notes, mais les commentaires sont anonymes, vous pouvez vous servir de l'espace réservé aux commentaires pour annoter les cartes (en n'oubliant pas de sauvegarder), tout est conservé sur une base de données pour usage ultérieur. Vous pouvez à tous moment changer d'idée, il suffit pour cela de ré-envoyer votre vote pour annuler le précédent. Ce site n'est pas responsable des propos qu'il permet de rendre accessible. Vous êtes responsable de vos propos. Merci de votre collaboration, pour toute question n'hésitez pas à me contacter : malric.philippe@gmail.com

Identification

Pour vous identifier, vous avez deux choix. Si vous avez un compte gmail, cliquez sur le G de Google, en haut à droite. Sinon cliquez sur le petit bonhome à côté pour vous enregistrer, un courriel vous sera envoyé pour être certain que c'est vous, répondez-lui. Ensuite, vous pourrez vous identifier (sur la page d'authentification).
